# スクリーンショット置き場

このディレクトリには、教材で使うVS CodeやGitHubのスクリーンショット画像を置きます。

現在入っている画像は、個人名・GitHubアカウント名・private repository名が写らないように作成した教材用スクリーンショットです。

## 優先して追加する画像

- `vscode-branch-main.png`: VS Code左下に `main` が表示されている状態
- `vscode-branch-work.png`: VS Code左下に作業ブランチが表示されている状態
- `vscode-source-control.png`: Source Controlで変更ファイルを確認する画面
- `github-pr-base-compare.png`: GitHubのPR作成画面で base と compare を確認する画面
- `vscode-conflict-buttons.png`: コンフリクト解消ボタンが表示されている画面

## 追加候補

- `vscode-create-branch.png`: `Create new branch` を選ぶ画面
- `vscode-stage-file.png`: 変更ファイル横の `+` ボタン
- `vscode-commit-message.png`: commitメッセージ欄と `Commit` ボタン
- `vscode-publish-branch.png`: `Publish Branch` を押す前の画面

## 撮影時の注意

- 個人名、メールアドレス、GitHubアカウント名を写さない
- 秘密情報やprivate repository名を写さない
- 必要なUIだけ切り取る
- クリックする場所を赤枠や番号で示す
- 危険操作のボタンは、押すことを促す画像にしない
